# Site images

## `hero-nz.jpg` (optional but recommended)

The homepage hero (`src/pages/index.tsx`) uses this file as a background photo,
layered under a dark navy gradient so white text stays readable.

**If this file is missing, the hero still looks polished** — it falls back to a
navy background plus an inline mountain-silhouette SVG. Drop the photo in and it
upgrades automatically; no code change needed.

### What to provide
- **File name:** `hero-nz.jpg` (exactly), placed in this folder.
- **Subject:** a wide New Zealand landscape — mountains, lake, or fjord.
- **Tone:** dark / dusk-toned works best (white hero text sits on top).
- **Size:** ~2400×1400px, landscape.
- **Weight:** keep under ~400 KB (compress the JPG).

No other image files are required — institute cards and icons are drawn with
CSS gradients and inline SVGs.
